    **Part B — Streaming route handler:**

    Create `src/app/api/reflection/run/route.ts` following the convergence snapshot route pattern but with streaming. The full implementation:

    1. `export async function POST(request: Request): Promise<Response>`
    2. Auth check: `const supabase = await createClient(); const { data: { user }, error: authError } = await supabase.auth.getUser();` Return 401 JSON if no user.
    3. Rate limit check: Query `reflection_sessions` for records with `created_at > NOW() - 24h`. If count > 0, return 429 JSON with `{ error: 'Reflection already run in the last 24 hours' }`.
    4. Context assembly (ALL before stream creation): Run parallel Supabase queries:
       - `supabase.from('nodes').select('id, title, node_type, status, description, author_id').neq('status', 'archived')`
       - `supabase.from('edges').select('source_id, target_id, edge_type')`
       - `supabase.from('tension_alerts').select('type, severity, description').eq('status', 'active')`
       - `supabase.from('convergence_snapshots').select('goal_space_id, score, computed_at').order('computed_at', { ascending: false }).limit(50)`
    5. Derive context from query results:
       - goalSpaces: filter nodes by node_type === 'goal_space'
       - triggerOutcomes: filter nodes by node_type === 'trigger_outcome'
       - activityByAuthor: reduce nodes to count per author_id (exclude nulls)
       - activeTensions: from tension_alerts query
       - convergenceSnapshots: from snapshots query
       - nodes: all non-archived nodes from query
    6. Build prompt: `const prompt = buildReflectionPrompt(ctx);`
    7. Count nodes for persistence: `const nodeCount = nodes.length;`
    8. Create ReadableStream:
       ```typescript
       const encoder = new TextEncoder();
       const { default: Anthropic } = await import('@anthropic-ai/sdk');
       const anthropic = new Anthropic({ apiKey: process.env.ANTHROPIC_API_KEY });

       const stream = new ReadableStream({
         async start(controller) {
           const messageStream = anthropic.messages.stream({
             model: process.env.REFLECTION_LLM_MODEL ?? 'claude-sonnet-4-20250514',
             max_tokens: 4096,
             system: REFLECTION_SYSTEM_PROMPT,
             messages: [{ role: 'user', content: prompt }],
           });

           let fullContent = '';
           for await (const chunk of messageStream) {
             if (chunk.type === 'content_block_delta' && chunk.delta.type === 'text_delta') {
               fullContent += chunk.delta.text;
               controller.enqueue(encoder.encode(chunk.delta.text));
             }
           }

           // Persist after stream completes
           try {
             const parsed = parseReflectionResponse(fullContent);
             await supabase.from('reflection_sessions').insert({
               machine_reflection: parsed,
               node_count_at_reflection: nodeCount,
               triggered_by: 'on_demand',
               run_by: user.id,
             });
           } catch {
             // Parse/persist failure should not break the stream
           }

           controller.close();
         },
       });
       ```
    9. Return: `new Response(stream, { headers: { 'Content-Type': 'text/plain; charset=utf-8', 'X-Content-Type-Options': 'nosniff', 'Cache-Control': 'no-cache' } })`

    Critical anti-patterns to avoid (from RESEARCH.md):
    - Do NOT create the ReadableStream before auth/rate-limit/context checks pass
    - Do NOT parse JSON mid-stream
    - Do NOT await the full response before streaming (use stream() not create())
