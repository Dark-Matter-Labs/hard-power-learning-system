    2. `computeOutcomeStatus(outcomeId, edges, allNodes)`:
       - Build a Map from allNodes for O(1) lookup: `new Map(allNodes.map(n => [n.id, n]))`
       - Filter edges where `target_id === outcomeId` (incoming edges)
       - Check blocked FIRST: any source node with `status === 'falsified'` or `status === 'suspended'` -> return 'blocked'
       - Check met: any `indicates_progress` edge whose source has `status === 'promoted'` -> return 'met'
       - Check in_progress: any `assigned_to_outcome` or `targets_outcome` edge exists -> return 'in_progress'
       - Default: return 'not_started'
       - IMPORTANT: Do NOT check `content.outcome` — that field does not exist until Phase 3

    3. `getOutcomeCommitmentCount(outcomeId, edges)`:
       - Return `edges.filter(e => e.edge_type === 'assigned_to_outcome' && e.target_id === outcomeId).length`

    4. `getOutcomeHunchCount(outcomeId, edges, allNodes)`:
       - Filter `targets_outcome` edges where `target_id === outcomeId`
       - Then filter to only those whose source node has `node_type === 'hunch'` (not intervention)
       - Return the count

    All functions MUST be pure (no side effects, no mutation). Use `readonly` parameter types.
